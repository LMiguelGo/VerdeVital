\doxysubsection{Descripción detallada}
Estructura para representar el estado de los actuadores. 

Estructura para almacenar el estado de los actuadores del invernadero.

Esta estructura contiene los estados de los actuadores como la bomba de agua, el ventilador y los LEDs. Cada actuador se representa con un valor de 0 (apagado) o 1 (encendido).
