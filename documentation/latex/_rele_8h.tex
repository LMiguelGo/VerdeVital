\doxysubsection{Descripción detallada}
Clase para controlar un relé 

Esta clase permite controlar un relé conectado a un pin específico del ESP32. Proporciona métodos para inicializar el relé y establecer su estado (activo o inactivo).
\begin{DoxyItemize}
\item El relé se activa con un valor lógico alto (HIGH) y se desactiva con un valor lógico bajo (LOW). 
\end{DoxyItemize}
