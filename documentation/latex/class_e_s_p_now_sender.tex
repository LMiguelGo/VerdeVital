\doxysubsection{Descripción detallada}
Clase para enviar datos de sensores usando ESP-\/\+NOW. 

Esta clase permite inicializar el envío de datos a un dispositivo específico utilizando ESP-\/\+NOW. Se encarga de configurar el canal y la dirección MAC del peer.
