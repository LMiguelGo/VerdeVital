\doxysubsection{Descripción detallada}
Clase para manejar sensores DHT (temperatura y humedad). Esta clase utiliza la biblioteca DHT para leer datos de temperatura y humedad. 

La clase \doxylink{class_sensor_d_h_t}{Sensor\+DHT} encapsula la funcionalidad de un sensor DHT, permitiendo inicializar el sensor, leer la temperatura y la humedad, y manejar los valores leídos. Los valores de temperatura y humedad se almacenan en variables privadas y se pueden acceder a través de métodos públicos.
