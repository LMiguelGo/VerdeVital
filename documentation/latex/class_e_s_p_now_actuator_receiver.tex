\doxysubsection{Descripción detallada}
Librerías. 

Clase para recibir estados de actuadores a través de ESP-\/\+NOW

Esta clase permite recibir estados de actuadores (bomba de agua, ventilador y LEDs) a través de ESP-\/\+NOW. Utiliza una estructura {\ttfamily \doxylink{struct_actuator_state}{Actuator\+State}} para representar el estado de los actuadores. El receptor se inicializa en un canal específico y permite registrar un callback para manejar los datos recibidos.
