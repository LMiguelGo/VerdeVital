\doxysubsection{Descripción detallada}
{\bfseries{PF-\/\+Edge}} es uno de los tres módulos principales del ecosistema {\bfseries{Verde\+Vital}}, diseñado para la gestión integral de invernaderos inteligentes. Este módulo actúa como el núcleo central del sistema, encargado de recibir, procesar y visualizar los datos provenientes de los sensores, así como de enviar comandos a los actuadores. Además, proporciona funcionalidades avanzadas como registro de datos, generación de alertas y comunicación con servicios externos como Telegram.\hypertarget{README.md_autotoc_md19}{}\doxysubsection{\texorpdfstring{Arquitectura del Sistema}{Arquitectura del Sistema}}\label{README.md_autotoc_md19}
El ecosistema {\bfseries{Verde\+Vital}} está compuesto por los siguientes módulos\+:


\begin{DoxyEnumerate}
\item {\bfseries{PF-\/\+Sensores\+:}} Captura datos ambientales (temperatura, humedad, luz, CO2, etc.) y los envía a PF-\/\+Edge mediante ESP-\/\+NOW.
\item {\bfseries{PF-\/\+Actuadores\+:}} Recibe comandos de PF-\/\+Edge para controlar dispositivos como bombas, ventiladores y LEDs.
\item {\bfseries{PF-\/\+Edge\+:}} Centraliza la información, procesa los datos, genera alertas y controla la interacción con los usuarios.
\end{DoxyEnumerate}\hypertarget{README.md_autotoc_md20}{}\doxysubsection{\texorpdfstring{Funcionalidades Principales}{Funcionalidades Principales}}\label{README.md_autotoc_md20}

\begin{DoxyItemize}
\item {\bfseries{Recepción de datos de sensores\+:}} Utiliza ESP-\/\+NOW para recibir datos de múltiples sensores ambientales conectados al módulo {\bfseries{PF-\/\+Sensores}}.
\item {\bfseries{Control de actuadores\+:}} Envía comandos a {\bfseries{PF-\/\+Actuadores}} para activar o desactivar dispositivos según los umbrales configurados.
\item {\bfseries{Gestión de umbrales y alertas\+:}} Evalúa los datos de los sensores en función de umbrales definidos, activa alertas y genera mensajes de estado.
\item {\bfseries{Registro de datos\+:}} Almacena los datos de los sensores en una tarjeta SD en formato CSV para análisis histórico.
\item {\bfseries{Interacción con Telegram\+:}} Permite enviar notificaciones y recibir comandos a través de un bot de Telegram.
\item {\bfseries{Visualización en pantalla OLED\+:}} Muestra información en tiempo real, como datos de sensores, estado de actuadores y alertas.
\end{DoxyItemize}\hypertarget{README.md_autotoc_md21}{}\doxysubsection{\texorpdfstring{Tecnologías y Plataforma}{Tecnologías y Plataforma}}\label{README.md_autotoc_md21}
