Estructura para almacenar datos de sensores. Esta estructura contiene campos para temperatura, humedad, luz, CO2, humedad del suelo y voltaje. 

Estructura para almacenar los datos de los sensores del invernadero.

\item humidity\+: Humedad relativa en porcentaje.
\item light\+: Valor de luz (0-\/65535).
\item co2ppm\+: Concentración de CO2 en partes por millón (ppm).
\item soil\+Moisture\+: Humedad del suelo en porcentaje.
\item voltage\+: Voltaje medido por el sensor.
