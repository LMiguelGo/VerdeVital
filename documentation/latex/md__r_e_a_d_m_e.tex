\chapter{Verde\+Vital -\/ Sistema Integral de Monitoreo y Control Ambiental}
\hypertarget{md__r_e_a_d_m_e}{}\label{md__r_e_a_d_m_e}\index{VerdeVital -\/ Sistema Integral de Monitoreo y Control Ambiental@{VerdeVital -\/ Sistema Integral de Monitoreo y Control Ambiental}}

{\bfseries{Verde\+Vital}} es un ecosistema modular diseñado para la gestión inteligente de invernaderos y entornos agrícolas. Este sistema combina tecnologías avanzadas de adquisición de datos, procesamiento, comunicación inalámbrica y control de actuadores para ofrecer una solución integral de monitoreo y automatización ambiental.\hypertarget{md__r_e_a_d_m_e_autotoc_md32}{}\doxysection{\texorpdfstring{Arquitectura del Sistema}{Arquitectura del Sistema}}\label{md__r_e_a_d_m_e_autotoc_md32}
El ecosistema {\bfseries{Verde\+Vital}} está compuesto por tres módulos principales que trabajan de manera coordinada\+:


\begin{DoxyEnumerate}
\item {\bfseries{PF-\/\+Sensores\+:}} Captura datos ambientales (temperatura, humedad, luz, CO2, humedad del suelo, voltaje) y los transmite al módulo central mediante ESP-\/\+NOW.
\item {\bfseries{PF-\/\+Edge\+:}} Actúa como el núcleo del sistema, procesando los datos recibidos, evaluando umbrales, generando alertas, registrando información y enviando comandos a los actuadores.
\item {\bfseries{PF-\/\+Actuadores\+:}} Recibe comandos del módulo central y controla dispositivos como bombas de agua, ventiladores y LEDs RGB.
\end{DoxyEnumerate}\hypertarget{md__r_e_a_d_m_e_autotoc_md33}{}\doxysection{\texorpdfstring{Características Principales}{Características Principales}}\label{md__r_e_a_d_m_e_autotoc_md33}
\hypertarget{md__r_e_a_d_m_e_autotoc_md34}{}\doxysubsection{\texorpdfstring{PF-\/\+Sensores}{PF-\/\+Sensores}}\label{md__r_e_a_d_m_e_autotoc_md34}

\begin{DoxyItemize}
\item {\bfseries{Adquisición de Datos\+:}} ~\newline
 Captura información de sensores como\+:
\begin{DoxyItemize}
\item {\bfseries{DHT11\+:}} Temperatura y humedad.
\item {\bfseries{LDR\+:}} Intensidad lumínica.
\item {\bfseries{MQ135\+:}} Calidad del aire (CO2 en ppm).
\item {\bfseries{YL-\/69\+:}} Humedad del suelo.
\item {\bfseries{Divisor de Voltaje\+:}} Voltaje de entrada.
\end{DoxyItemize}
\item {\bfseries{Comunicación Inalámbrica\+:}} ~\newline
 Transmite los datos al módulo {\bfseries{PF-\/\+Edge}} mediante ESP-\/\+NOW.
\item {\bfseries{Multitarea con Free\+RTOS\+:}} ~\newline
 Ejecuta tareas concurrentes para la lectura de sensores y envío de datos.
\end{DoxyItemize}\hypertarget{md__r_e_a_d_m_e_autotoc_md35}{}\doxysubsection{\texorpdfstring{PF-\/\+Edge}{PF-\/\+Edge}}\label{md__r_e_a_d_m_e_autotoc_md35}

\begin{DoxyItemize}
\item {\bfseries{Procesamiento Centralizado\+:}} ~\newline
 Recibe datos de {\bfseries{PF-\/\+Sensores}}, evalúa umbrales configurados y genera alertas.
\item {\bfseries{Control de Actuadores\+:}} ~\newline
 Envía comandos a {\bfseries{PF-\/\+Actuadores}} para activar o desactivar dispositivos según las condiciones ambientales.
\item {\bfseries{Registro de Datos\+:}} ~\newline
 Almacena información en una tarjeta SD en formato CSV para análisis histórico.
\item {\bfseries{Interacción con Telegram\+:}} ~\newline
 Permite a los usuarios consultar el estado del sistema y controlar actuadores mediante un bot de Telegram.
\item {\bfseries{Visualización en Pantalla OLED\+:}} ~\newline
 Muestra datos en tiempo real, alertas y estado del sistema.
\end{DoxyItemize}\hypertarget{md__r_e_a_d_m_e_autotoc_md36}{}\doxysubsection{\texorpdfstring{PF-\/\+Actuadores}{PF-\/\+Actuadores}}\label{md__r_e_a_d_m_e_autotoc_md36}

\item {\bfseries{Configuración Modular\+:}} ~\newline
 Permite la expansión para incluir más actuadores según las necesidades del sistema.
