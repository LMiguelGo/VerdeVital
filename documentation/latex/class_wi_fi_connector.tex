\doxysubsection{Descripción detallada}
Clase para gestionar la conexión Wi\+Fi. 

Proporciona métodos para conectarse a una red Wi\+Fi y obtener detalles de la conexión, como el canal, la dirección IP, el SSID y la intensidad de la señal.
