\doxysubsection{Descripción detallada}
Este repositorio contiene el código fuente de {\bfseries{PF-\/\+Actuadores}}, uno de los tres proyectos fundamentales del sistema integral {\bfseries{Verde\+Vital}}. Este proyecto se encarga de controlar los actuadores (bomba de agua, ventilador y LEDs RGB) de manera remota mediante el protocolo ESP-\/\+NOW, comunicándose con el dispositivo central {\bfseries{PF-\/\+Edge}}.\hypertarget{README.md_autotoc_md1}{}\doxysubsection{\texorpdfstring{Arquitectura del Sistema}{Arquitectura del Sistema}}\label{README.md_autotoc_md1}
El sistema {\bfseries{Verde\+Vital}} se compone de tres proyectos principales\+:


\begin{DoxyEnumerate}
\item {\bfseries{PF-\/\+Sensores\+:}} Captura datos ambientales y los envía al dispositivo central.
\item {\bfseries{PF-\/\+Edge\+:}} Recibe la información de los sensores, procesa los datos y envía comandos a los actuadores.
\item {\bfseries{PF-\/\+Actuadores\+:}} Este proyecto, que recibe los comandos de PF-\/\+Edge y controla físicamente los dispositivos (bomba, ventilador, LEDs).
\end{DoxyEnumerate}\hypertarget{README.md_autotoc_md2}{}\doxysubsection{\texorpdfstring{Funcionamiento General}{Funcionamiento General}}\label{README.md_autotoc_md2}

\item {\bfseries{Configurar el ESP32 en el Arduino IDE\+:}} ~\newline

 Abra \href{http://_vscodecontentref_/0}{\texttt{ main.\+cpp}}, seleccione el puerto y placa correctos para su ESP32 y cargue el proyecto.

\item Asigne el hardware a los pines indicados\+:

\item Ventilador\+: Pin 26.
\item LEDs RGB\+: Pines 14, 27, 33 y 32.
\end{DoxyItemize}
\end{DoxyItemize}
