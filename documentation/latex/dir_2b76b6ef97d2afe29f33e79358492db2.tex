\doxysubsection{Descripción detallada}
PF-\/\+Sensores es uno de los tres módulos principales del ecosistema Verde\+Vital. Este proyecto se encarga de la adquisición de datos de múltiples sensores ambientales mediante un ESP32, utilizando Free\+RTOS para gestionar tareas concurrentes y ESP-\/\+NOW para transmitir datos de manera inalámbrica. La información recogida se integra con los módulos PF-\/\+Actuadores y PF-\/\+Edge para ofrecer una solución integral de monitoreo y control ambiental.\hypertarget{README.md_autotoc_md10}{}\doxysubsection{\texorpdfstring{Características}{Características}}\label{README.md_autotoc_md10}

\begin{DoxyItemize}
\item {\bfseries{Lectura de Sensores Ambientales}} ~\newline
 Se capturan datos de\+:
\begin{DoxyItemize}
\item {\bfseries{DHT\+:}} Medición de temperatura y humedad (por ejemplo, DHT11).
\item {\bfseries{LDR\+:}} Detección de intensidad lumínica (valor analógico entre 0 y 1023).
\item {\bfseries{MQ135\+:}} Evaluación de la calidad del aire mediante la medición de CO2 (ppm).
\item {\bfseries{YL-\/69\+:}} Medición de la humedad del suelo, convertida a un valor porcentual.
\item {\bfseries{Sensor de Voltaje\+:}} Lectura del voltaje presente en un divisor de voltaje.
\end{DoxyItemize}
\item {\bfseries{Estructura de Datos Unificada}} ~\newline
 Todos los valores se agrupan en la estructura definida en {\ttfamily \doxylink{sensor_data_8h}{sensor\+Data.\+h}}, que almacena\+:
\begin{DoxyItemize}
\item Temperatura (ºC)
\item Humedad (\%)
\item Luz
\item CO2 (ppm)
\item Humedad del suelo (\%)
\item Voltaje
\end{DoxyItemize}
\item {\bfseries{Comunicación Inalámbrica con ESP-\/\+NOW}} ~\newline
 La clase {\ttfamily \doxylink{class_e_s_p_now_sender}{ESPNow\+Sender}} (en {\ttfamily ESPNow\+Sender.\+h}) gestiona la configuración y el envío de los datos mediante ESP-\/\+NOW, estableciendo el modo Wi\+Fi en STA, configurando el canal de comunicación y registrando callbacks para el monitoreo del estado de envío.
\item {\bfseries{Multitarea con Free\+RTOS}} ~\newline
 Se ejecuta una tarea periódica (cada 3 segundos) que\+:
\begin{DoxyItemize}
\item Lee los datos de cada sensor.
\item Los almacena en la estructura {\ttfamily \doxylink{struct_sensor_data}{Sensor\+Data}}.
\item Envía la información al nodo receptor configurado.
\end{DoxyItemize}
