Descripción detallada

Clase para manejar el registro de datos en una tarjeta SD.

Esta clase permite inicializar la tarjeta SD, crear directorios y archivos, y registrar datos de sensores en formato CSV.

Registra datos de sensores en un archivo CSV.

Crea los directorios necesarios y escribe los datos en un archivo CSV con encabezados si el archivo está vacío.

Parámetros
timestampMarca de tiempo en formato "YYYY-MM-DD HH:MM:SS".
nodeIdIdentificador del nodo que envía los datos.
rssiIntensidad de la señal WiFi en dBm.
dataEstructura con los datos de los sensores.
systemStateEstado del sistema (por ejemplo, 0 para normal, 1 para alerta).
Devuelve
true si los datos se registraron correctamente, false en caso de error.
