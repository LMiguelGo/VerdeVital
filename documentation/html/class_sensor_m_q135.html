Descripción detallada

Clase para manejar el sensor MQ135 (calidad del aire). Esta clase permite leer el valor de CO2 del sensor MQ135 conectado a un pin analógico.

La clase SensorMQ135 encapsula la funcionalidad de un sensor MQ135, permitiendo inicializar el sensor y leer el valor de CO2 en partes por millón (ppm).
