Descripción detallada

Sistema de automatización VerdeVital usando ESP32 y actuadores.

Este archivo contiene la implementación del sistema de automatización VerdeVital, que utiliza un ESP32 para controlar diversos actuadores como bombas, ventiladores y LEDs. El sistema recibe comandos a través de ESP-NOW y controla los actuadores.

Los actuadores y sus pines respectivos son:

  • Bomba de agua: Pin 25
  • Ventilador: Pin 26
  • LEDs RGB: Pines 14, 27, 33, 32

También se implementa ESP-NOW para la comunicación entre dispositivos (Edge y Actuadores).
