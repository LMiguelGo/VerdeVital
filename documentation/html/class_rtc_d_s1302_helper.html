Descripción detallada

Clase para interactuar con el módulo RTC DS1302.

Esta clase proporciona métodos para inicializar, configurar y obtener datos del reloj en tiempo real DS1302.

◆ begin()

void RtcDS1302Helper::begin ( )
inline

Inicializa el módulo RTC DS1302.

Configura el módulo, desactiva la protección de escritura, inicia el reloj si no está corriendo y actualiza la fecha/hora desde el tiempo de compilación si es necesario.
