Descripción detallada

Clase para enviar datos usando ESP-NOW en ESP32. Esta clase permite enviar datos de sensores a través de ESP-NOW. La estructura de datos enviada es idéntica a SensorData.

Nota
Asegúrate de incluir la estructura SensorData en tu proyecto. Esta clase está diseñada para ser utilizada en un entorno ESP32.
