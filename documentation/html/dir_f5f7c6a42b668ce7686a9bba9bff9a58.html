 Clase para enviar datos usando ESP-NOW en ESP32. Esta clase permite enviar datos de sensores a través de ESP-NOW. La estructura de datos enviada es idéntica a SensorData.

 Clase para manejar sensores DHT (temperatura y humedad). Esta clase utiliza la biblioteca DHT para leer datos de temperatura y humedad.

 sensor_MQ135.h
 Clase para manejar el sensor MQ135 (calidad del aire). Esta clase permite leer el valor de CO2 del sensor MQ135 conectado a un pin analógico.
 
 sensor_Voltage.h
 Clase para manejar sensores de voltaje. Esta clase permite leer el voltaje de un divisor de voltaje conectado a un pin analógico.
 
 sensor_YL69.h
 Clase para manejar el sensor de humedad YL-69. Esta clase permite leer el nivel de humedad del suelo utilizando el sensor YL-69.
 
 sensorData.h
 Estructura para almacenar datos de sensores. Esta estructura contiene campos para temperatura, humedad, luz, CO2, humedad del suelo y voltaje.
