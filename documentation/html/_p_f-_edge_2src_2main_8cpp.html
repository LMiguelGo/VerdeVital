Descripción detallada

Programa principal para el sistema de monitoreo y control de invernadero.

Este programa utiliza ESP-NOW para recibir datos de sensores, controla actuadores, se comunica con un bot de Telegram y registra datos en una tarjeta SD. Utiliza FreeRTOS para manejar múltiples tareas concurrentes.
